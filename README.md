# 立川ナイトガイド（tachikawa-night-guide）

立川駅周辺のキャバクラ・ガールズバー3店舗の紹介＋求人まとめの静的サイト。
**safety-road.jp（Road リポジトリ）とは完全に独立した別サイト**（掲載3店舗から正式依頼を受けて制作）。

## 構成

| ページ | 内容 |
|---|---|
| `index.html` | トップ（3店舗の紹介＋求人導線） |
| `jobs.html` | **求人まとめ**（3店舗の募集情報を集約・応募は各店へ直接） |
| `chess.html` | CLUB CHESS（キャバクラ）紹介＋求人 |
| `candydrop.html` | Candy Drop（ガールズバー）紹介＋求人 |
| `chicchi.html` | Girl's Bar chicchi（ガールズバー）紹介＋求人 |

## 法令・ポリシー上の注意（変更禁止）

- **18歳以上（高校生不可）** の表記を全ページ維持（労基法61条: 22時以降の深夜業は18歳以上のみ）
- **20歳未満の飲酒禁止** の注意書きを維持
- 性的サービスを想起させる表現・「誰でも高収入」等の誇大表現は使わない
- 雇用契約は応募者と各店舗の直接契約（当サイトは募集情報の掲載のみ）
- 求人情報を他社サイトから転載しない（掲載は依頼店舗の提供情報のみ）

## 広告出稿の方針（決定済み）

- 予算: **1,000円/日**
- **セーフティロードサービスの Google 広告アカウントからは出さない**（業種混在・ポリシーリスク隔離のため**別アカウント新設**）
- ナイトワーク求人は Google 広告ポリシー（アルコール・雇用関連）で不承認になりやすい。審査落ちした場合の代替: X 広告・ナイトワーク専門求人媒体・SNS運用
